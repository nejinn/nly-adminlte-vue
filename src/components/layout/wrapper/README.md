# wrapper

## nly-container-wrapper

此组件会修改body的class,props参数自由搭配会有不同的布局,请自由发挥

### props

参数 | 类型 |  默认值 | 描述
-|-|-|-
sideMini | Boolean | 无 | 边侧栏是否可以收起,true可以收起,false将边侧画板左侧滑入消失
layout | String | 无 | 整体布局,可选fixed和boxed
navbarFixed | Boolean | 无 | 头部导航fixed布局
footerFixed | Boolean | 无 | 底部fixed布局
topNav | Boolean | 无 | 头部导航顶格无边侧栏布局
